UNIT 2
Reading & Writing

📖 Words in Context

These questions ask: "As used in the text, what does [X] most nearly mean?" — They test context, NOT vocabulary memorization.

The 4-Step Strategy

  1. Cover the word — blank it out mentally
  2. Read around it — sentence + surrounding context
  3. Predict your own word before reading choices
  4. Match to the best answer — closest to your prediction
⚠️ Trap: The test often includes the word's most common definition as a wrong answer. The word is almost always used in a more specific contextual way.

Common Trap Types

UNIT 3
Reading & Writing

✏️ Text Completion & Precision

Format: blank in a sentence — "Which choice completes the text with the most logical and precise word or phrase?"

Strategy: CLUE → PREDICT → MATCH

  1. Find the clue — what does the surrounding text signal belongs in the blank?
  2. Predict — form your own word before reading choices
  3. Match precisely — right answer is BOTH logical AND the most specific fit
⚠️ Two answers may both "work" — the SAT wants the MOST precise one.

Signal Word Patterns

Contrasthowever, but, despite, although, yet, whileOpposite of adjacent clause
Continuationin fact, moreover, additionally, furthermoreSame direction as stated idea
Cause-effectbecause, since, therefore, thus, as a resultLogical consequence or explanation
Elaborationfor example, such as, including, specificallyMore specific version of main claim

Rhetorical Synthesis Questions

  • Given: student's notes (bullet points)
  • Task: complete a sentence using those notes as evidence
  • Strategy: read the stem → find the supporting note → eliminate choices that contradict notes or add outside info
UNIT 4
Reading & Writing

🔍 Hard Comprehension Questions

Command of Evidence — Textual

Format: "Which quote from the text best supports the claim that..."

  • Find the quote that directly proves the specific claim
  • Related ≠ right. The quote must prove the exact claim being made

Command of Evidence — Quantitative

  • Includes a table, graph, or chart
  • Read the claim precisely → find the exact matching data point
  • Close ≠ correct — the data must precisely match the claim

Inference Questions

  • ✅ Inference must be directly supported by the text — no outside knowledge
  • ✅ "Must be true" test: if the text is true, must this answer be true?
  • ❌ Eliminate extreme answers ("always," "never") unless the text explicitly says so
  • ❌ Eliminate answers that go beyond what the text says

Text Structure & Purpose

Describe the function (why it's there), not the content (what it says).

Common functions: introduce a claim, provide evidence, present a counterargument, illustrate with an example, qualify a statement, show a contrast

Cross-Text Connection Questions

  1. Summarize Passage 1 in one sentence
  2. Summarize Passage 2 in one sentence
  3. Identify the relationship: agree / disagree / one builds on the other
  4. Answer ONLY from the text — no background knowledge
💡 For hard questions: go back to the passage FIRST, form your own answer, THEN read the choices. Prevents answer choices from muddying your thinking.

🔢 Algebra, Linear Functions & Systems

Linear Functions

  • Slope-intercept: y = mx + b (m = slope, b = y-intercept)
  • Slope: m = (y₂ − y₁) / (x₂ − x₁) = Δy/Δx
  • x-intercept: set y = 0, solve for x
  • Slope from table: constant rate of change for linear functions

Systems of Equations

TypeConditionLines
One solutionDifferent slopesIntersect at one point
No solutionSame slopes, different y-interceptsParallel
Infinite solutionsSame slopes AND same y-interceptsIdentical

Absolute Value Equations

  • |A| = c → two cases: A = c OR A = −c
  • Combine like terms with |same expression| before splitting cases
  • Example: 7|4 − x| = 49 → |4 − x| = 7 → x = −3 or x = 11
For "no solution" problems: set slopes equal and solve for the unknown parameter.

🧮 Polynomials & Factoring

Factor Theorem

(x − c) is a factor of f(x) if and only if f(c) = 0. To test a factor: substitute and check result.

Key Factoring Patterns

PatternFormulaExample
Difference of Squaresa² − b² = (a+b)(a−b)x² − 9 = (x+3)(x−3)
Perfect Squarea² ± 2ab + b² = (a ± b)²x² + 6x + 9 = (x+3)²
Trinomial (a=1)(x+p)(x+q): p+q=b, pq=cx² − 2x − 15 = (x−5)(x+3)
Trinomial (a≠1)AC method or grouping3x²+20x−63 = (x+9)(3x−7)

Rational Expressions

Factor numerator and denominator completely, then cancel common factors.

Example: (x²−2x−15)/(x²−9) = (x−5)(x+3) / (x+3)(x−3) = (x−5)/(x−3)

Division Identity Trick

If P(x)/(tx−a) = Q(x) + R/(tx−a), then P(x) = Q(x)·(tx−a) + R. Multiply out and match coefficients to find unknown constants.

⚙️ Functions & Composite Functions

Composite Functions

f(g(x)) — Work inside → out. Evaluate g(x) first, then plug into f.

f(g(a)) ≠ g(f(a)) — order matters.

Key Function Concepts

ConceptMeaningSAT Application
f(g(x))Apply g first, then fSubstitute g(x) for every x in f
f(a) = 0a is a zero/rootGraph crosses x-axis at (a, 0); (x−a) is a factor
Rate of changeSlope of linear functionFor f(x) = mx + b, rate = m
