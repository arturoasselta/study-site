Synthetic Division & Rational Zeros
  • Synthetic Division: Divide poly by (x−c); if remainder = 0, c is a zero
  • Remainder Theorem: f(c) = remainder when f(x) ÷ (x−c)
  • Factor Theorem: (x−c) is a factor ↔ f(c) = 0
Possible Rational Zeros (PRZ)
PRZ = ±(factors of constant) / (factors of leading coeff)
Sign graph: plot zeros on number line, test intervals for +/−

Piecewise Functions
  • Evaluate using the rule whose domain condition matches x
  • Continuity: pieces must connect at boundaries
  • Closed dot (•): point included; Open dot (○): excluded
  • Check limits from left & right at each break point
Exam Tips
  • Factor before canceling — never cancel unfactored terms
  • State domain restrictions before simplifying rational expressions
  • Verify inverse: f(f⁻¹(x)) = x AND f⁻¹(f(x)) = x
  • PRZ ≠ actual zeros — must verify with synthetic division
  • Expand log of products; condense sums of logs back to one log
  • Radians vs degrees — know which mode your calculator is in!
